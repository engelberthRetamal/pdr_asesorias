Descripción del Proyecto

El objetivo de este proyecto es desarrollar una solución tecnológica que mejore la gestión, el control, la seguridad y la disponibilidad de información vital tanto para la empresa como para sus clientes. Esto busca resolver los problemas de desorganización de información y la pérdida de tiempo en los procesos, lo que a su vez mejorará la productividad.

El sistema requerido permitirá la gestión de capacitaciones, brindando la posibilidad de crear nuevas capacitaciones. Además, se gestionarán tres tipos de usuarios: clientes, administrativos y profesionales. La aplicación permitirá la creación y edición de estos usuarios, almacenando todos los datos en una base de datos llamada "prevencion_riesgos" en MySQL.

**********

Descripción de la Aplicación Web

La aplicación web consta de las siguientes páginas:

    -Página de inicio: Proporciona un resumen de las funcionalidades disponibles en la aplicación.
    -Página de contacto: Permite a los usuarios enviar mensajes directos.
    -Página de creación de usuario: Permite crear un nuevo usuario.
    -Página de creación de capacitación: Permite crear una nueva capacitación.
    -Página de listado de usuarios: Muestra un listado de todos los usuarios existentes.
    -Página de listado de capacitaciones: Muestra un listado de todas las capacitaciones existentes.
    -Página de inicio de sesión (login): Permite a los usuarios iniciar sesión en la aplicación para hacer uso de las funcionalidades de esta.
    -Opción de cierre de sesión (logout): Permite cerrar la sesión previamente iniciada.

Toda la aplicación web es responsiva, lo que significa que se puede utilizar en cualquier dispositivo, ya que la vista de cada página se adapta automáticamente a la pantalla del dispositivo utilizado.
